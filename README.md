# Nexus: City Events Finder

## Description
Nexus is a web application that allows users to search for upcoming events in a city. It aggregates data from multiple APIs and displays them in a unified interface.

## Features
- Multi-API integration (Ticketmaster, SerpApi, PredictHQ)
- Load balancing of results
- Duplicate removal
- Cloud deployment (Google Cloud Run)
